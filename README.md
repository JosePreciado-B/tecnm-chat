# TecNM Chat – Monorepo (Frontend + Backend)

Este proyecto contiene la estructura base para una app de chat tipo WhatsApp,
pensada para el Tecnológico Nacional de México (colores azul/blanco).

- `frontend/`: App móvil con **React Native + Expo + TypeScript** (para usar con Expo Go).
- `backend/`: API Node + Express + TypeScript con datos **mock** (solo estructura, sin base de datos).

> Nota: El login en el frontend **no es funcional todavía**,
> solo navega a la lista de chats usando datos mock.  
> El backend está listo para que otra persona lo conecte a **Supabase** más adelante.
